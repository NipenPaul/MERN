** Download the MongoDB community server package

.deb file for ubuntu 22.04
https://www.mongodb.com/try/download/community

** Open the Terminal Ctrl+Alt+T

1. sudo dpkg -i mongodb-org-server_7.0.1_amd64.deb
2. sudo systemctl status mongod
3. sudo systemctl start mongod


** Download the MongoDB shell community
https://www.mongodb.com/try/download/shell

1. sudo dpkg -i fileName.deb
sudo dpkg -i mongodb-mongosh_2.0.0_amd64.deb

Terminal: mongosh
Terminal: show dbs
